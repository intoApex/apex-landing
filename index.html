Deploy in 30 minutes

Tested on Ubuntu 24.04, DigitalOcean and Hetzner. The guide covers every step.

terminal — apex deployment
# Install OpenClaw + copy APEX configs
sudo npm install -g openclaw && openclaw onboard
cp SOUL.md.template ~/.openclaw/workspace/SOUL.md
cp -r skills/* ~/.openclaw/skills/
cp -r config/* ~/.openclaw/workspace/config/

# Verify bootstrap is optimized (target: under 3K bytes)
wc -c ~/.openclaw/workspace/*.md
 2070 total    ← 80% smaller than default ✓

# Enable memory search (free Gemini embeddings)
openclaw memory index --force
 11 files · 18 chunks · gemini ready ✓

# Create autonomous cron jobs
openclaw cron add --name "daily-post" \
  --light-context --thinking off --no-deliver
 ✓ System running. Modules fire on schedule.

A VPS ($4-48/month depending on provider), an Anthropic API key, and 30 minutes. The deployment guide covers every step. Works on Ubuntu 24.04, tested on DigitalOcean and Hetzner.
How much will API costs be?
With APEX's routing and caps: $7-15/month for all modules. Without optimization, the same workload costs $50-200+. Per-module budget caps prevent surprises — when a module hits its cap, it pauses and alerts you.
Can't I just write these SKILL.md files myself?
The individual skills, sure. The integrated system with model routing, per-module spending caps, bootstrap trim (2,335 bytes vs 12,000 default), memory architecture with semantic search, anti-spam safeguards, compaction with memory flush, and cron flags that cut costs 30-50% — that took us 5 days of trial, error, and optimization. You're buying the lessons, not just the files.
Does social media work without X Premium?
Yes, but with 280-char tweets and lower visibility. X Premium ($8/month) gives 4x visibility and 4,000-char tweets. The module includes the Premium patch. Cold replies are blocked by the X API regardless — the module uses quote tweets as a workaround we discovered on Day 3.
What if I get stuck?
The deployment guide covers every error we hit: 403 cold replies, dollar sign interpolation, memory search unavailable, credit errors, cron timeouts. For anything else, DM @IntoApex — we respond to every message.
